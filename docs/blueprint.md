# **App Name**: ConversionFlow

## Core Features:

- Hero Section Animation: Animated background elements with parallax effect in the hero section to capture attention.
- Dynamic Services Tabs: Interactive tabs for Web Development, App Development, and UI/UX Design & Branding that dynamically change the content below.
- Web Development Cards Grid: Display website categories (e.g., Coaching, Gym & Fitness) with key benefits and starting prices, including hover animations. In cards show example website in every category usinfg html css
- App Development Cards Grid: Showcase app categories (e.g., Ecommerce, Food Delivery) with modern app mockups, feature highlights, and 'Request Demo' CTAs. In every category show an example app of the category box using html and css
- Interactive Pricing Plans: Comparison table for Starter, Business, and Premium tiers with feature highlights and shine/glow animations. Show prices according to country's currency.
- Portfolio Showcase Carousel: Carousel of past work with live links, case study previews, and technology tags.
- Contact Form and WhatsApp Integration: Modern contact form (Name/Email/Business Type/WhatsApp) and a floating WhatsApp button for easy lead capture.

## Style Guidelines:

- Primary color: Neon green (#39FF14) to convey innovation and energy.
- Background color: Dark charcoal (#121212) with a desaturated neon green overlay at 20% saturation.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body, providing a modern and readable aesthetic.
- Use minimalistic, line-based icons with a neon green accent.
- Dark theme with glassmorphism cards and smooth, spacious design.
- Smooth scroll transitions, stagger fade-ins, hover tilt on cards, and shine/glow animation on pricing cards to enhance user experience.
